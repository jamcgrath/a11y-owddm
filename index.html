What is A11y

Dictionary: Easy to reach, use, understand

Who is A11Y for?

  • Blind People?
  • Deaf People?
  • People who live where the law mandates A11y?

A11Y is for everybody!

Why?

A11Y benefits everyone

  • Physical Disabilities
  • Visual Disabilities
  • Auditory Disabilities
  • Cognitive Disabilities
  • Temporary Disabilities

physical = missing limbs, quadriplegia - paralyzed from neck down, arthritis, parkinsons

Visual = blind, reduced vision, color blindness

Auditory = deaf or profound hearing loss

Cognitive = Memory, ADHD, Difficulty with reading and communication understanding

Temporary = Broken limbs, Migraines, Pregnancy, Hangover

When should you add A11y

From the very start.

It's never too late to do it.*

Developing and designing with A11Y in mind makes the web a place where anyone regardless of location, situation, abilities, language, etc... can interact with others to learn and share their knowledge and experiences.

Anyone should be able to use any website!

How?

Good question

  • Semantic HTML
  • Use Aria roles, attributes, landmarks, etc...
  • Testing with tools like AXE browser plugin
talk about how there are patterns but it's not so simple sometimes.
